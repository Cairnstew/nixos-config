# Repository Structure

Annotated tree of all non-trivial files in the repository, grouped by directory.

## Root Configuration Files

Core flake and repository metadata files.

| File | Description |
|------|-------------|
| `flake.nix` | Main flake definition with inputs and outputs wiring via flake-parts |
| `config.nix` | User identity configuration (imported by modules/flake-parts/config.nix) |
| `template.nix` | Template for creating new NixOS host configurations |
| `AGENTS.md` | Top-level policy for agents working on this repository |
| `README.md` | Human-readable project overview |
| `justfile` | Just command runner recipes |
| `.envrc` | direnv configuration for development shell |
