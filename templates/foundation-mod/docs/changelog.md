## Foundation 1.5.10.0131

### API Major Changes

- New COMP_QUOTABLE component
- New DESIRABILITY_LEVEL enumeration

### Asset exposition

Numerous core assets have been exposed to modding:

- Study behavior tree
- Scholar job
- Study room building function
- Roman church blueprint
- Village center building part, and building function
- Hop farm building, building part, and building function
- Brewery building, building part, and building function
- Multiple new villager status
- Happiness factors
- Default house manager data
- Default village center resource collection
- All unlockables (buildings, building part lists, assignable building functions, trade bonuses, and walls)
- Horse vehicle
- Villager need types

## Foundation 1.5.7.0123

### Global Mods Changes

Mods distributed without their generated_ids.lua now fail to load.

### API Major Changes

- Enumerations can now be used as more than just strings
- Custom Components
- Character Setup refactored into CHARACTER_SETUP
- Complex building zones with BUILDING_ZONE
- Military: MILITARY_QUEST_GENERATOR, MILITARY_CAMPAIGN, DATA_MILITARY_MISSION, SOLDIER_STATE, WEAPON
- Game rules: GAME_RULE

### API Minor Changes

Various property additions and changes across BALANCING, BUILDING, BUILDING_CONSTRUCTOR, BUILDING_FUNCTION_FORESTER, BUILDING_FUNCTION_STUDY_ROOM, HAPPINESS_FACTOR, JOB, QUEST_FUNCTION, VEHICLE_ANIMATION, VILLAGER_STATUS, WEATHER_SYSTEM

## Foundation 1.4.5.1009

- Create custom maps
- Building part functions are now assets, declared individually
- Core building part functions have also been exposed to the API
- A resource can now have multiple resource types
- Prefabs for the game can now be exposed
- The market tender behavior asset is now available
- New buildings, building parts, resources and jobs assets are now available
- The resource container component is available

## Foundation 1.3.1.0802

- FBX material override
- Building part lists are replaced with building part set lists
- Event choices can now trigger multiple actions
- EVENT_CONDITION and its derived types have been renamed and are backwards compatible
