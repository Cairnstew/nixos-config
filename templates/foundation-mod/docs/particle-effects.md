# Particle Effects

PARTICLE_SYSTEM asset configures a certain type of particle system.

COMP_PARTICLE_EMITTER component is attached to a game object and emits the particle system in the scene.

COMP_PARTICLE_EMITTER_TOGGLE is a particle emitter controller that stops and restarts the COMP_PARTICLE_EMITTER when disabled/enabled.
